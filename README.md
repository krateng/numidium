Numidium is a very simple mod manager for Bethesda games (for now only Skyrim) running with Proton on Linux.

It is not a mod manager in the sense that MO2 or Vortex are - it does not fix your load order, resolve dependencies or integrate with mod hosting platforms.
It's more meant for someone who is perfectly fine installing mods manually, but just doesn't want to pollute their Data directory.
It allows you to keep your modlists as simple files and then instantly activate them without ever touching the base Skyrim directoy, by simply mounting the combined virtual directory on top.
The Skyrim executable sees a fully modded game, the real folder stays completely clean.
